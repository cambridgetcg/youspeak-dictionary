# YOUSPEAK Dictionary

_Words the world doesn't have yet._

A living dictionary of 151 constructed words from 12 ancient tongues —
naming what modern languages cannot easily say. A cathedral of vocabulary.
A forge of missing words. A standing worship.

## What this is

YOUSPEAK forges missing words from Hebrew, Greek, Sanskrit, Akkadian,
Egyptian, Yoruba, Mandarin, Japanese, Sumerian, Avestan, Arabic, and more —
combining them into compact words that name what existing modern languages
cannot easily say.

This is the internet's living dictionary of those words. Searchable,
filterable, open. No gatekeeping.

## Run locally

```bash
npm install
npm run dev      # http://localhost:3001
npm run build    # static export to out/
```

## Deploy

Static export — host anywhere. GitHub Pages, Codeberg Pages, Netlify,
Vercel, anywhere that serves HTML.

## The 5 tiers

- **Core** (113) — everyday YOUSPEAK, all six constitutional foundations met
- **Specialized** (18) — domain-specific vocabulary
- **Mathema** (9) — mathematics-grounded words, the dual-way bridge
- **Theobasis** (3) — the deepest ground, Creator-Created relationship
- **Convergence** (8) — cross-tradition convergences
